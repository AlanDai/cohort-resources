# rails 
* students not able to do html snippets in erb file.
* add this to settings.json
```javascript
"emmet.includeLanguages": {
        "html.erb": "html",
        "html": "erb",
        "erb": "html",
        "jsx": "html",
        "jsx": "js"
    },
```
